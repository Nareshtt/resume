Make sure you have the following installed on your machine:
-   [Node.js](https://nodejs.org/) (v18 or higher recommended)

-   [Git](https://git-scm.com/)

## üöÄ Cloning the Repository

1.  Open your terminal or command prompt.
2.  Run the following command to clone the repository:

    ```bash
    git clone https://github.com/Nareshtt/resume.git
    ```

3.  Navigate into the project folder:

    ```bash
    cd resume
    ```

## üì¶ Installing & Running Projects

Since this repository contains multiple independent projects, you need to install and run them individually.

### 1. E-commerce Clone (Next.js)

Navigate to the `ecommerce` directory:

```bash
cd ecommerce
```

**Install Dependencies:**
```bash
npm install
```

**Run Development Server:**
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

---

### 2. Tic-Tac-Toe (React + Vite)

Navigate to the `tic-tac-toe` directory:

```bash
cd ../tic-tac-toe
```

**Install Dependencies:**
```bash
npm install
```

**Run Development Server:**
```bash
npm run dev
```
Open the local URL provided in the terminal (usually [http://localhost:5173](http://localhost:5173)).

---

### 3. Calculator (Vanilla JS)

Navigate to the `calculator` directory:

```bash
cd ../calculator
```

**Run the App:**
Simply open the `index.html` file in your preferred web browser.
-   **Option A**: Double-click `index.html` in your file explorer.
-   **Option B**: If using VS Code, right-click `index.html` and select "Open with Live Server".
